## 🎬 Interaction States

### Button States
```
NORMAL:
┌──────────────────┐
│ ▶ Start Design   │ bg: #3b82f6 (blue)
└──────────────────┘

HOVER:
┌──────────────────┐
│ ▶ Start Design   │ bg: #06b6d4 (cyan) - smooth transition
└──────────────────┘

ACTIVE/PRESSED:
┌──────────────────┐
│ ▶ Start Design   │ bg: darker variant
└──────────────────┘
```
- 200ms animation duration
- Smooth color transitions
- Clear visual feedback

### Status Indicator Animation
```
Connected (Pulsing):
● ─ ─ ─ ─   (full opacity, "Connected" text visible)
  ↓ (fade over 0.75s)
◐ ─ ─ ─ ─   (50% opacity)
  ↓ (fade over 0.75s)
● ─ ─ ─ ─   (back to full)
```
- Pulses every 1.5 seconds
- Attracts attention without being jarring

## 📐 Layout Dimensions

- Width: 1600px

- Scalable and responsive

- Left Panel: 420px (Design inputs)

### Element Heights
- Top Bar: 60px
- Chat Header: 64px
- Chat Input: 80px
- Buttons: 48px (primary), 40px (secondary)
- Cards: 56-60px typically
- Input Fields: 44-48px

### Spacing
- Panel margins: 16-20px
- Between elements: 8-18px depending on grouping
- Border width: 1px (subtle)
- Border radius: 6-12px (smooth)

- **11px** - Secondary text

## ✨ Visual Effects

### Shadows & Depth
- Subtle borders (1px, `#475569`)
- No hard shadows
- Layered appearance through color gradients
- Rounded corners add depth

### Transparency
- Most elements at 100% opacity
- ComboBox background at 60% for subtle effect
- Pulsing indicators use opacity transitions

### Animations
- **Button hover** - 200ms color transition
- **Status pulse** - 1.5s loop (fade in/out)
- **Smooth scrolling** - ListView with clip
- **No janky movements** - Professional feel
